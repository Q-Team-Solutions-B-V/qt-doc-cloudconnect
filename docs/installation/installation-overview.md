# Installation Overview

Cloud Connect can be installed from Microsoft AppSource. The app is compatible with Microsoft Dynamics 365 Business Central SaaS deployments.

## Prerequisites

- Microsoft Dynamics 365 Business Central (SaaS)
- Access to Azure Blob Storage (for cloud storage functionality)

## Installation Steps

1. Obtain the Cloud Connect app from Microsoft AppSource or your solution provider.
2. Follow the standard Business Central extension installation process.
3. Assign the required permission sets to users who will manage or use external storage.
4. Complete the initial configuration as described in the configuration guide.

For detailed configuration, see [Configuration](configuration.md).
