# Configuration

After installing Cloud Connect, configure the app to connect to your preferred external storage provider.

## Initial Setup

1. Open the Cloud Connect Assisted Setup page in Business Central.
2. Follow the guided steps to configure Azure Blob Storage or another supported provider.
3. Enter the required credentials and storage container information.
4. Test the connection to ensure proper setup.

## Permission Sets

Assign the `CloudConnect` permission set to users who need to manage or access external storage features.

## Assisted Setup

The app provides an Assisted Setup wizard for easy configuration and onboarding.
