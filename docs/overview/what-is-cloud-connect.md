# What is Cloud Connect?

Cloud Connect is an extension for Microsoft Dynamics 365 Business Central that enables seamless external storage of business documents and attachments using cloud providers such as Azure Blob Storage. The app optimizes database performance by storing files outside the Business Central database, while maintaining a familiar user experience for attaching and accessing documents.

## Key Features

- Store attachments in external cloud providers (Azure Blob Storage)
- Seamless integration with standard Business Central attachment workflows
- Document type routing — assign document types to attachments to control which provider(s) they are stored in
- Store the same file in multiple providers simultaneously
- Automatic re-routing when routing rules change
- Easy download and restore of externally stored files
- Optimized storage and database performance

## Technical Specifications

- Available for Business Central SaaS
- Compatible with Azure Blob Storage
- OneDrive support coming in a future version

## Included

- 24/7 Online support
- Automatic updates (SaaS only)
