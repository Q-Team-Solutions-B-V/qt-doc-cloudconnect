# What is Cloud Connect?

Cloud Connect is an extension for Microsoft Dynamics 365 Business Central that enables seamless external storage of business documents and attachments using cloud providers such as Azure Blob Storage. The app optimizes database performance by storing files outside the Business Central database, while maintaining a familiar user experience for attaching and accessing documents.

## Key Features

- Store attachments in external cloud providers
- Seamless integration with standard Business Central workflows
- Support for Azure Blob Storage
- Easy uploading and downloading of attachments
- Optimized storage and database performance
- Retrieve all stored files from external storage
