# Frequently Asked Questions

**Q: Which storage providers are supported?**
A: Azure Blob Storage is supported. But the app can be extended to support other providers. OneDrive support is coming soon.

**Q: How do I assign permissions?**
A: Assign the `CloudConnect` permission set to all users who need access to external attachments.

**Q: How can I set up my own storage account?**
A: You can connect your own Azure Blob Storage account during the assisted setup integrated in Business Central. Search for "Set Up External File Accounts".

**Q: Where can I get support?**
A: Contact Q-Team Solutions via the website or support channels listed in the documentation.
