# Usage Overview

Cloud Connect works seamlessly with standard document attachment workflows in Microsoft Dynamics 365 Business Central. Users can attach, view, and manage documents as usual, while the app handles external storage in the background.

## Attaching Documents

- Attach files to business records using the standard Business Central interface.
- Files are automatically stored in the configured external cloud provider.

## Accessing Attachments

- Users can view and download attachments directly from Business Central.
- The app retrieves files from external storage as needed.

## Assisted Setup

Use the Assisted Setup wizard for guided onboarding and configuration.
